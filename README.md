# Quantized Rupture Cosmology

This repository contains the LaTeX source, simulation code, and supporting
materials for the paper:

**"A Quantized Rupture Cosmology: Cyclic Dynamics, Multiverse Branching, and Entropy Inheritance"**  
*Paul Jarvis (Independent Researcher, Much Hadham, UK)*

## Overview

This work introduces a rupture-driven cyclic cosmology in which black hole
formation increases a singularity density variable until a critical threshold is
reached. Crossing this threshold triggers a rupture that ends the cycle and
produces a quantized number of descendant universes.

The model is defined by a single compact equation:



\[
N(t) = 6\,\Theta(\rho_s(t) - \rho_{\rm crit})\,m(\Delta(t)),
\]



where the excitation mode \(m\) is determined by discrete overshoot intervals.
Simulations show that the fundamental mode \(m=1\) overwhelmingly dominates,
yielding six universes per rupture.

- `paper/main.pdf` — compiled version
